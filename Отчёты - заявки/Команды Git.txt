git fetch извлечь (fetch) всю информацию, которая есть в репозитории Павла, но нет в вашем

git pull, как правило, извлекает (fetch) данные с сервера, с которого вы изначально склонировали, и автоматически пытается
слить (merge) их с кодом, над которым вы в данный момент работаете.
